# Spanish Ground

A personal, modular Spanish-learning playground. Lesson packs can be expanded from tutor notes, vocabulary, phrases, texts, memes, or any topic discussed with Codex.

## Local development

```bash
npm install
npm run dev
```

## Adding a lesson

Structured lesson content lives in `src/content/lessonPacks.js`. Every pack can power flashcards, quizzes, sentence building, and contextual dialogue practice.

Pushing to `main` automatically builds and deploys the app to GitHub Pages.
